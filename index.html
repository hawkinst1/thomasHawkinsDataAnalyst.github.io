Lessons

Portfolio projects are great but at a beginner's level I often forget the basic stuff and so I have put together some lessons on certain subjects that I found tricky to do or that I find myself forgetting as a reference to my journey in learning data analytics.

Splitting text and numbers from a single column | 26th August 2023

In creating some data about myself I downloaded my Fitbit data and took three categories: Date, Steps and Calories. The first two are simple and are a data and integer, but for calories the website helpfully gave me the following format n cals.

Coming from a PHP background, I immediately thought of the substring function, but this is not available. Instead you can interact with the columns directly and split based on their content. In this case "digit to non-digit"

Errors in the data | 28th August 2023

When going through the fitbit data I downloaded, I noticed that instead of dd/mm/2022 it was giving me dd/mm/2023. My challenge now was to change the correct data to 2022.

I found the going to Python made things much easier, although setting it up for my VS Code took a little troubleshooting now it is set up transforming the dates into the format I needed was fairly simple.

I think this may be the faster method for transforming data and would probably opt for it over SQL or Power Query Editor

QuickDBD | 29th August 2023

In doing some research and exploring tutorials I have found QuickDBD which creates SQL tables based on your input allowing you to see them in realtime and to create relations before exporting to SSMS and acutally creating the schema in SQL.
